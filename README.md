# 💼 AI Job Recommender System
**Capstone Rekayasa Data 2026 — Universitas Siliwangi**

Sistem rekomendasi lowongan kerja berbasis **Hybrid RAG** (Retrieval-Augmented Generation) yang menggabungkan:
- **SBERT** untuk pencarian semantik berbasis vektor
- **Google Gemini** sebagai lapisan presentasi percakapan

---

## 🏗️ Arsitektur Sistem

```
User Input (Natural Language)
        │
        ▼
┌───────────────────────┐
│   Gemini (Extractor)  │  → Ekstrak: skills, lokasi, gaji, remote
└───────────────────────┘
        │
        ▼
┌───────────────────────┐
│   SBERT Vector Search │  → Cosine Similarity (70% bobot)
└───────────────────────┘
        │
        ▼
┌───────────────────────┐
│  Filtering & Ranking  │  → Gaji (15%) + Lokasi/Remote (15%)
└───────────────────────┘
        │
        ▼
┌───────────────────────┐
│  Gemini (Humanizer)   │  → Rangkum hasil jadi jawaban natural
└───────────────────────┘
        │
        ▼
     Output Chat
```

### Logika Pembobotan
| Faktor | Bobot | Jenis |
|--------|-------|-------|
| Skill Similarity (SBERT) | **70%** | Soft Score |
| Kesesuaian Gaji | **15%** | Bonus Score |
| Lokasi & Remote | **15%** | Hard Filter |

## ⚡ Cara Menjalankan

### 1. Install Dependensi

```bash
pip install -r requirements.txt
```

### 2. Siapkan API Key Gemini

Buat file `.env` di root folder:
```
GEMINI_API_KEY=AIza_your_key_here
```

Dapatkan API key gratis di: https://aistudio.google.com/app/apikey

### 3. Jalankan Preprocessing (SEKALI SAJA)

```bash
python src/preprocess.py
```

> Proses ini akan membuat file `models/job_embeddings.pkl` (~15-20 MB).
> Waktu sekitar 2-5 menit tergantung CPU.

### 4. Jalankan Aplikasi

```bash
streamlit run app.py
```

Buka browser ke: **http://localhost:8501**

## 📊 Dataset

Dataset yang digunakan dalam penelitian ini adalah `job_data.csv`, yang memuat metrik pekerjaan global lintas industri seperti *Quantum Computing*, *Green Tech*, *Blockchain*, *Data Science*, dan lainnya. 

Dataset ini terdiri dari 9 kolom yang mencakup informasi detail mengenai posisi, kompensasi, hingga kualifikasi pendidikan. Seluruh data kualifikasi keahlian (*skills*) dipusatkan ke dalam **satu kolom teks tunggal** untuk mempermudah prapemrosesan data pada tahap pemodelan.

### Kamus Data (Data Dictionary)

| Nama Kolom | Tipe Data | Deskripsi |
| :--- | :--- | :--- |
| `job_title` | Kategorikal | Nama posisi atau peran pekerjaan (contoh: *Quantum Researcher*, *Data Analyst*). |
| `industry` | Kategorikal | Sektor industri tempat posisi tersebut berada (contoh: *Consulting*, *Telecom*, *Finance*). |
| `salary_usd` | Numerik | Estimasi kompensasi gaji tahunan dalam mata uang Dolar AS (USD). |
| `skills_required` | Teks | Kumpulan seluruh keahlian teknis dan *tools* yang diwajibkan untuk posisi tersebut (format dipisahkan oleh koma). |
| `remote_option` | Kategorikal | Kebijakan kehadiran kerja yang ditawarkan perusahaan (*Remote*, *Onsite*, atau *Hybrid*). |
| `company_size` | Kategorikal | Skala atau ukuran perusahaan (*Small*, *Medium*, *Large*). |
| `education_level` | Kategorikal | Syarat tingkat pendidikan minimal (contoh: *Bachelor's Degree*, *Master's Degree*, *PhD*, *Diploma*). |
| `city` | Kategorikal | Kota lokasi penempatan kerja. |
| `country` | Kategorikal | Negara lokasi penempatan kerja. |

## ⚠️ Troubleshooting

| Masalah | Solusi |
|---------|--------|
| `Engine belum siap` | Jalankan `python src/preprocess.py` |
| `GEMINI_API_KEY belum diatur` | Buat file `.env` atau masukkan di sidebar |
| `ModuleNotFoundError` | Jalankan `pip install -r requirements.txt` |
| Hasil tidak relevan | Pastikan query menggunakan bahasa Inggris untuk skill teknis |
